# Story Draft Checklist

Use by @sm when creating a new story draft.

## Structure
- [ ] Story uses correct template
- [ ] Epic ID and story number assigned
- [ ] Title follows convention: verb + noun
- [ ] Status set to Draft

## Content
- [ ] User story format: As a... I want... So that...
- [ ] Acceptance criteria: Given/When/Then format
- [ ] Scope section has IN and OUT items
- [ ] Dependencies listed (or "None")

## Metadata
- [ ] Complexity estimate included
- [ ] Business value stated
- [ ] Risks noted (or "None identified")
