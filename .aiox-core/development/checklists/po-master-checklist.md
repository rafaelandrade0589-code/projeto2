# PO Master Checklist

Use during story validation (validate-story-draft).

## Story Quality (10-point)
- [ ] 1. Clear and objective title
- [ ] 2. Complete description (problem/need explained)
- [ ] 3. Testable acceptance criteria
- [ ] 4. Well-defined scope (IN and OUT)
- [ ] 5. Dependencies mapped
- [ ] 6. Complexity estimated
- [ ] 7. Business value clear
- [ ] 8. Risks documented
- [ ] 9. Definition of Done clear
- [ ] 10. Alignment with PRD/Epic confirmed

## Decision
- [ ] Score >= 7: GO — update status to Ready
- [ ] Score < 7: NO-GO — list required fixes
